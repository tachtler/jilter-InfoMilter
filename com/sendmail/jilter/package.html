Main package for Jilter. Filters implemented with Jilter typically implement the {@link com.sendmail.jilter.JilterHandler} interface. The Milter protocol itself is handled by the {@link com.sendmail.jilter.JilterProcessor} class. Simple standalone Jilter. The {@link com.sendmail.jilter.samples.standalone.SimpleJilterServer} class implements a simple standalone filter, which you can use with your own classes that implement the {@link com.sendmail.jilter.JilterHandler} interface.
